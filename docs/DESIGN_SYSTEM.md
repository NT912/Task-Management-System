# Design System & Brand Identity

Tài liệu quy định hệ thống thiết kế và nhận diện thương hiệu cho dự án **TeamsTask**.

## 1. Color Palette (Bảng Màu Mới)

| Tên Màu | Mã Hex | Vai trò (Role) | Sử dụng (Usage) |
| :--- | :--- | :--- | :--- |
| **Ocean Blue** | `#0071bc` | **Primary (Chủ đạo)** | Logo Text, Nút chính (Sign up, Submit), Tiêu đề. |
| **Sky Blue** | `#29abe2` | **Secondary** | Hover state, Link phụ. |
| **Lime Green** | `#8cc63f` | **Brand Accent** | Màu nhận diện thương hiệu, icon, điểm nhấn. |
| **Forest Green** | `#009245` | **Success / Text** | Thông báo thành công, Link quan trọng. |
| **White** | `#ffffff` | **Background** | Nền trang. |
| **Black** | `#000000` | **Text** | Văn bản chính. |
